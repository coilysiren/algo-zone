# algo-zone

pronounced like "auto zone", a place for figuring out algorithms

## Installation requirements

- python
- pipenv
- docker

## Usage

Run this once

```bash
pipenv sync
```

Then run any of these as many times as you want

```bash
# 1st example
pipenv run tests

# 2nd example
pipenv run tests $language

# 3rd example
pipenv run tests rust
```
